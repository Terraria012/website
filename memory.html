RAMとは何か？

RAMとはRandom Access Memory の略称であり、コンピュータの記憶装置の一つである。
RAMは揮発性メモリ(電源を落とすとデータが消えてしまうメモリ)であるが、epromやpromなどと違い理論上無限回書き込みが行える。

どうやってRAMを作るか思考する

8byteメモリを作るには次のことを考えなくてはいけない。
  • どのICを使うのか
  • 何に配線するのか
  • 規模はどのぐらいになるのか

ICについて

メモリとして使うことができるICは以下の3つである。
  • RS-latch
  • D-latch
  • D-flipflop
  • JK-flipflop
今回はこれらの回路の中でも一番扱いやすいD-flipflop(個人の感想です)を使っていこうと思う。
汎用ロジックICにおいて、74HC74がD-flipflop回路にあたる。
74HC74にはD-flipflopが2ユニット入っているため、8byteメモリを作るのに必要な74HC74の数は
8(byte)×8(bit)/2(ユニット)=32(個)

配線と規模について

8byteメモリを配線するつまり、32個の74HC74を配線するのは簡単ではなく、回路の規模も大きくなる。
とりあえず4bitメモリをユニバーサル基板上で再現する。
実際に配線して感じたことは大変すぎ!!この一言に尽きる。
4bitの配線を組むだけでほぼ8時間かかった。さらに、必要な基板数はおよそ8枚。 CPU製作において、メモリだけで8段も使ってしまうと今ある基板だけでは足りず、 追加で購入しなくてはならないうえに、配線だけで多くの時間を浪費してしまう。 そこで小型化と配線の簡略化とロマンのために、空中配線を採用することにした。

空中配線していく

筐体を作る

空中配線するのはいいけど、何にメモリを取り付けるのか。 基板に取り付けては、空中配線の意味が無いし、空中配線したものをさらに空中配線するのは地獄を見ることになる。
では、どうしたらいいだろうか。
そうだ、筐体を作ろう
メモリはLEDで光るから、その光が見やすいように筐体の柱は細めに、でも強度は必要…
